# Japanese Martial Arts — Grappling History — Index

## Brief
An interactive page exploring Japanese martial arts history (712 CE – 1964) through primary sources. Standalone tradition page within the Grappling History project, alongside Fiore (Italian) and Ancient Greece (Pankration). Built as a static HTML/CSS/JS page driven by `data.json`, following the vertical fishbone timeline pattern established by the Ancient Greece page. Reworked from an existing horizontal fishbone prototype (`grappling_fishboneJapaneseHistory.html`) that covers the full grappling lineage — this project narrows scope to Japanese traditions only.
